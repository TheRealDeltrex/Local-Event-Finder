📍 Local Events Finder

Find things to do near you. Pick a spot, set a range, and mark events as date ideas or family friendly.

In-browser demo. The interface is fully live here — location, distance, tagging and filtering all work. But browsers can't fetch the event sites directly (no CORS), so when a live scrape isn't reachable this demo falls back to sample events near your location. For real listings, install the Android app or run the local Flask app (see the home page).
