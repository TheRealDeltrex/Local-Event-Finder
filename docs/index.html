📍 Local Events Finder

Find things to do near you. It scrapes local events, works out how far each one is, and tags them as date ideas or family-friendly — so you can filter down to exactly what you're after.

🤖 Android app

A native app with device GPS, offline tag storage, and the same date / family tagging. Signed by Deltrex.

Download the APK

Android 8.0+. Enable “install from unknown sources”, then open the file.

Run the web version locally (recommended)

The in-browser demo is handy, but the local Flask app is faster and more reliable (it scrapes server-side, so there's no CORS proxy in the way).

git clone https://github.com/TheRealDeltrex/Local-Event-Finder
cd Local-Event-Finder
pip install -r requirements.txt
python app.py            # opens http://127.0.0.1:5001

What it does

  • Web scraping — reads schema.org event data from configurable sources (Meetup, AllEvents), so one extractor works across many sites.
  • Your location — device GPS or a typed town/city, resolved via OpenStreetMap.
  • Within range — every event is geocoded and distance-filtered (up to 40 km).
  • Tags — auto-tagged as possible date / family friendly, with manual overrides that stick.
  • Filter — narrow results live by tag and range.
